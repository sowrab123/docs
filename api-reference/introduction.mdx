# CP Platform API Documentation

Welcome to the CP Platform API reference. This API allows users to manage problems, bookmarks, statuses, and authentication records for a competitive programming platform.

- 🌐 Base URL: `https://cp-platform-backend.vercel.app`
- 🧪 Local URL: `http://127.0.0.1:8000`

Use the left navigation to explore endpoints grouped by method (GET, POST, DELETE, etc).
